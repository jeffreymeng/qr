Welcome to QRID

A qr reading and writing service mainly intended for internal development and use only, but you're still welcome to use it if you want.

Documentation:

To generate a qr code, include a iframe with the following url: 

https://qr.jeffkmeng.com/generate?param1name=param1value&param2name=param2value&param3name=param3value

Please note that all parameters must be URL encoded if they are not url safe. You can use this tool. Parameters:
Parameter Description Defualt
data The data, URL encoded, that the qr code should show. required

Optional if you choose link for the widget attribute(if you use this attribute, the link text will be changed to this attribute). Ignored if you did not choose html or text for widget. null

URL encoding tool:

This is encoded with the standard encodeURIComponent() function of javascript.

    • What are correction levels?

    ECC (Error Correction Capacity) levels represent the amount of data that can be restored if the QR code symbol is smudged or damanged. There are four ECC(Error Correction Capacity) levels ranging from L (smallest) to H (best)
    L - Low : About 7% of codewords can be restored
    M - Medium : About 15% of codewords can be restored
    Q - Quartile : About 25% of codewords can be restored
    H - High : About 30% of codewords can be restored
